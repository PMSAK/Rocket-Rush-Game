# 🚀 Rocket Escape

> A 3D Unity game where you pilot a rocket through increasingly dangerous obstacle courses using precise movement, thrust, and dash mechanics.

---

## 📖 About

Rocket Escape is a physics-based 3D arcade game developed in **Unity** using **C#**.

Your mission is simple: guide the rocket safely to the finish platform without colliding with obstacles. While the objective remains the same, every level becomes progressively more difficult by introducing new obstacle patterns, tighter spaces, and faster decision-making.

The game contains **5 handcrafted levels**, each designed to test the player's control, timing, and reaction speed.

This project was built to improve my understanding of Unity fundamentals, game physics, scripting, particle systems, scene management, and gameplay programming.

---

## ✨ Features

- 🚀 Smooth rocket movement
- 🔥 Realistic thrust mechanics
- ⚡ Dash system for quick horizontal movement
- 💥 Particle effects for:
  - Rocket thrust
  - Dash
  - Explosion
  - Level completion
- 🎯 Five progressively difficult levels
- 🛰️ Multiple obstacle types
  - Static obstacles
  - Linear moving obstacles
  - Random movement obstacles
- 🎮 Responsive controls
- 🌄 Low-poly Mars-inspired environment
- 🏁 Finish sequence for every level

---

## 🎮 Controls

| Key | Action |
|------|--------|
| **A** | Move Left |
| **D** | Move Right |
| **Space** | Thrust Up |
| **Q** | Dash Left |
| **E** | Dash Right |
